# Форум Neo-Tokyo

Этот модуль содержит все компоненты для форума проекта Neo-Tokyo.

## Структура

```
forum/
├── ForumPage.tsx              # Главная страница форума
├── ForumPage.module.scss      # Стили главной страницы
├── CreateTopicPage.tsx        # Страница создания новой темы
├── CreateTopicPage.module.scss # Стили страницы создания темы
├── TopicPage.tsx             # Страница просмотра темы с комментариями
├── TopicPage.module.scss      # Стили страницы темы
├── index.ts                   # Экспорт всех компонентов
└── README.md                  # Этот файл
```

## Компоненты

### ForumPage
Главная страница форума, отображает:
- Статистику форума (количество тем, сообщений, пользователей, просмотров)
- Категории форума с описаниями и статистикой
- Список последних тем с тегами (NEW, HOT, PINNED)
- Поиск по форуму
- Кнопку создания новой темы

### CreateTopicPage
Страница создания новой темы с формой:
- Заголовок темы (обязательное поле)
- Выбор категории (обязательное поле)
- Содержание темы (обязательное поле)
- Теги (необязательное поле)

### TopicPage
Страница просмотра темы с:
- Полным содержанием темы
- Статистикой (лайки, дизлайки, просмотры)
- Списком комментариев с возможностью лайка/дизлайка
- Формой добавления нового комментария
- Поддержкой ответов на комментарии

## Роуты

- `/forum` - Главная страница форума
- `/forum/create-topic` - Создание новой темы
- `/forum/topic/:id` - Просмотр темы по ID

## Стили

Все стили написаны в киберпанк-стиле с использованием:
- Неоновых цветов (синий, розовый, фиолетовый)
- Темного фона
- Анимаций свечения
- Градиентов и теней

## Использование

```tsx
import { ForumPage, CreateTopicPage, TopicPage } from './pages/forum';

// В роутах
{
  path: '/forum',
  Component: ForumPage,
}
```
